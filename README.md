### Declaración de casos de negocio

#### Caso 1: Plataforma meteorológica municipal

La Municipalidad de Villa Ficticia contrata a la consultora para desarrollar una aplicación web orientada al análisis y seguimiento meteorológico.

El sistema cruza datos provenientes de los principales servicios meteorológicos mediante APIs, incorpora registros históricos y presenta información meteorológica en tiempo real.

Además, la aplicación genera reportes cada 3 horas utilizando diferentes modelos especializados en climatología.

#### Caso 2: Aplicación comunitaria para asistencia en cultivo

La consultora decide desarrollar una solución con impacto social en la comunidad donde opera, ampliando también su alcance social.

La aplicación permite que el usuario indique su localización y, a partir de esa información, genera un esquema de cultivo suponiendo que la persona comenzara a plantar ese mismo día.

Las recomendaciones son generadas mediante IA, considerando la ubicación y las condiciones disponibles.

#### Caso 3: Sistema de monitoreo para energía sustentable

Una empresa de energías sustentables necesita monitorear el consumo eléctrico de viviendas y la energía generada por paneles solares.

La consultora desarrolla un sistema encargado de almacenar, ordenar y presentar los datos obtenidos.

Los cálculos técnicos y el análisis especializado quedan a cargo de los ingenieros de la empresa. El sistema actúa como plataforma de registro, organización y visualización de información.

#### Caso 4: Paneles de monitoreo y datos bursátiles

La consultora recibe el encargo de desarrollar paneles de monitoreo utilizando ThingsBoard.

Además, se le solicita integrar otras gráficas que obtienen datos bursátiles y los muestran en tiempo real.

El objetivo del sistema es centralizar información dinámica y presentarla mediante paneles visuales de consulta y seguimiento.

#### Caso 5: Sistema de consulta documental sobre NAS mediante RAG

Una empresa tiene problemas para localizar archivos dentro de su sistema de archivado en un NAS.

La consultora desarrolla un sistema de consulta basado en RAG, que permite buscar y localizar archivos a partir de consultas en lenguaje natural o criterios asociados al contenido documental.

El objetivo es facilitar la recuperación de información y mejorar el acceso a los archivos almacenados.
